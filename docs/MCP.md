# MCP-сервер EdgeLab Space

EdgeLab Space публикует библиотеку сообщества как MCP-сервер. Подключите его в
любом MCP-клиенте — Claude Code, cowork, Hermes, Cursor — и ваш агент получит
прямой доступ к навыкам, разборам, дропам и дайджестам, не открывая интерфейс.

## Подключение

Два способа — выбирайте по клиенту.

### Способ 1 — кастомный коннектор (OAuth, без ручного ключа)

Для **Claude.ai** (веб и десктоп) и **Claude Cowork**. Добавляете EdgeLab Space
как кастомный коннектор по одному URL — ключ вводить не нужно, доступ
подтверждается входом через Telegram.

1. Настройки → «Коннекторы» → «Добавить кастомный коннектор».
2. URL: `https://mcp.edgelab.space/mcp`
3. Клиент сам зарегистрируется (OAuth 2.1, Dynamic Client Registration) и откроет
   вход. Войдите через бота `@edgelabspacebot` и нажмите «Подтвердить».
4. На экране согласия разрешите доступ — клиент получит токен и подключится.

Доступ привязан к вашей подписке: при отмене подключение перестаёт работать.
Токены хранятся на сервере в зашифрованном виде; подключение можно отозвать.

### Способ 2 — персональный ключ (Bearer)

Для **Claude Code, Codex, Cursor** и обычного `curl` — там, где коннектор
задаётся конфигом.

| Параметр | Значение |
|---|---|
| URL | `https://mcp.edgelab.space/mcp` |
| Транспорт | Streamable HTTP (stateless, JSON-ответы) |
| Аутентификация | `Authorization: Bearer edgelabspace_...` |

Персональный ключ формата `edgelabspace_...` выпускается в кабинете на
`platform.edgelab.space/agent-identity`. Ключ показывается один раз — сохраните
его. Доступ действует, пока активна ваша подписка; при отмене сервер начинает
отвечать `Unauthorized`.

## Инструменты

Все инструменты только для чтения и возвращают материалы, доступные вашей
когорте.

| Инструмент | Назначение |
|---|---|
| `list_library(category?, limit)` | Список материалов библиотеки (category: lesson / skill / usecase / live / workshop / guide) |
| `search_library(query, limit)` | Поиск по библиотеке по названию и описанию |
| `get_material(slug)` | Полный материал: цель, результат, статья, копируемый промпт скилла, видео, таймкодный транскрипт, шаги, дочерние уроки |
| `list_hub_questions(query?, category?, limit, offset)` | Лента вопросов сообщества (AgentTinder) |
| `get_hub_question(question_id)` | Полный вопрос + источники, файлы и комментарии/ответы |

## Пример конфигурации клиента

### Claude Code (`.mcp.json`)

```json
{
  "mcpServers": {
    "edgelab-space": {
      "type": "http",
      "url": "https://mcp.edgelab.space/mcp",
      "headers": {
        "Authorization": "Bearer edgelabspace_YOUR_KEY"
      }
    }
  }
}
```

### Универсальный MCP-клиент

Любой клиент, поддерживающий remote MCP по Streamable HTTP, подключается тем же
URL и заголовком `Authorization`. Сервер работает stateless: каждый вызов —
самостоятельный HTTP-запрос, без отдельного хендшейка сессии.

## Проверка

После подключения попросите агента вызвать `list_categories()` — пустой или
непустой список категорий подтвердит, что ключ принят и подписка активна. Если
сервер отвечает `Unauthorized`, проверьте ключ и статус подписки.

> Нет MCP-клиента? Тот же сервер вызывается обычным `curl` по JSON-RPC —
> см. [CLI.md](CLI.md).
